# Contributing to AI Automation Projects

First off, thank you for considering contributing! It's people like you that make these automation tools better for everyone.

## How Can I Contribute?

### Reporting Bugs
* Check the existing issues to ensure the bug hasn't been reported yet.
* Open a new issue with a clear title and a detailed description of the problem.

### Suggesting Enhancements
* Open an issue to discuss your idea.
* Describe the specific use case and why the enhancement would be beneficial.

### Pull Requests
1. Fork the repository.
2. Create a new branch for your feature or bugfix.
3. Commit your changes with clear descriptions.
4. Push to your fork and submit a pull request.

## Technical Standards
* Ensure your n8n workflow JSON files are properly formatted.
* Keep your AI prompt templates clean and well-documented.
* Use meaningful node names within the n8n canvas for readability.

Thank you for your support!
